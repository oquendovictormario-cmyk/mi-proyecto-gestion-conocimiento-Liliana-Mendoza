# Proyecto Dashboard de Inteligencia Estratégica

## Descripción del Proyecto
Este proyecto realiza un análisis bibliométrico y de sentimientos sobre publicaciones científicas en gestión del conocimiento e innovación educativa, cubriendo el período 2022-2026.

## Hallazgos Principales
- **Total de publicaciones analizadas:** 1,336
- **Tendencia:** Decreciente (R² = 0.828)
- **Sentimiento predominante:** Positivo (89.9%)

## Herramientas Utilizadas
- Google Colab (Jupyter Notebook)
- Python con librerías: pandas, matplotlib, scikit-learn, vaderSentiment, wordcloud
- GitHub para control de versiones

## Contenido del Repositorio
- `Base de datos (1).xls` - Datos bibliográficos (1,336 registros)
- `Untitled0.ipynb` - Cuaderno con todo el análisis y código
- `README.md` - Descripción del proyecto

## Resultados del Análisis

### Análisis de Sentimientos (NLP)
- 89.9% artículos POSITIVOS
- 9.4% artículos NEGATIVOS
- 0.8% artículos NEUTRALES

### Predicción (Machine Learning)
- Modelo: Regresión polinomial (grado 2)
- Precisión: R² = 0.828
- Proyección: Disminución continua de publicaciones
